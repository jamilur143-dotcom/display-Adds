প্রজেক্ট ডেপ্লয়মেন্ট এবং ফায়ারবেস সেটআপ চিত্রসহ গাইডলাইন

গিটহাব পাবলিশিং, ভার্সেল হোস্টিং এবং ফায়ারবেস ক্লাউড ডাটাবেস সেটআপের প্রতিটি ধাপের ছবিসহ পূর্ণাঙ্গ গাইডলাইন।

📁 পার্ট ১: GitHub Desktop দিয়ে কোড গিটহাবে আপলোড (পাবলিশ) করা

আপনার কম্পিউটারের লোকাল প্রজেক্ট ফোল্ডারটি অনলাইনের গিটহাব অ্যাকাউন্টে নিরাপদে সেভ করার সহজ নিয়ম নিচে দেওয়া হলো:

ধাপ ১ GitHub Desktop সফটওয়্যার ওপেন করা

সফটওয়্যারটি চালু করুন। শুরুতে আপনার সামনে এরকম উইন্ডো দেখাবে যেখানে কোনো প্রজেক্ট সিলেক্ট করা থাকবে না অথবা পূর্বের কোনো প্রজেক্ট থাকবে।

ধাপ ২ লোকাল প্রজেক্ট ফোল্ডার অ্যাড করা

ওপরের বাম কোণায় File মেনু থেকে Add local repository... অপশনে ক্লিক করুন। তাহলে নিচের মতো পপআপ আসবে।

ধাপ ৩ প্রজেক্টের ফোল্ডার লোকেশন সিলেক্ট করা

Choose... বাটনে ক্লিক করে আপনার ড্রাইভ থেকে প্রজেক্ট ফোল্ডারটি D:\softare\display Adds সিলেক্ট করুন।

ধাপ ৪ নতুন রিপোজিটরি তৈরি করার সতর্কতা

যেহেতু প্রজেক্টটি আগে কখনো গিটহাবে আপলোড করা হয়নি, তাই লাল বর্ডারে একটি সতর্কবার্তা দেখাবে। সেখানে থাকা নীল রঙের create a repository লিঙ্কে ক্লিক করুন।

ধাপ ৫ গিট ইগনোর (Git ignore) ফাইল সিলেক্ট করা

রিপোজিটরি তৈরির ফর্মে প্রজেক্টের নাম দিন এবং **Git ignore** ড্রপডাউন মেনু থেকে **Node** সিলেক্ট করুন। এর ফলে ভারী ও অপ্রয়োজনীয় ফোল্ডারগুলো গিটহাবে আপলোড হবে না। এরপর **Create repository**-তে ক্লিক করুন।

ধাপ ৬ রিপোজিটরি পাবলিশ বাটন

রিপোজিটরি ক্রিয়েট হওয়ার পর স্ক্রিনের মাঝখানে নীল রঙের **Publish repository** বাটনটি চলে আসবে। সেটিতে ক্লিক করুন।

ধাপ ৭ প্রজেক্ট পাবলিক করা (টিক মার্ক তুলে দেওয়া)

এখানে সবচেয়ে গুরুত্বপূর্ণ কাজ হলো **Keep this code private** লেখার পাশে থাকা নীল রঙের टिक চিহ্নটি **তুলে দেওয়া (আনচেক করা)**। এর ফলে প্রজেক্টটি লাইভ হোস্ট করা যাবে। এরপর **Publish repository** বাটনে ক্লিক করুন।

🚀 পার্ট ২: Vercel দিয়ে সম্পূর্ণ ফ্রিতে ওয়েবসাইট লাইভ করা

গিটহাবের কোডটি স্বয়ংক্রিয়ভাবে একটি লাইভ ওয়েবসাইটে রূপান্তর করার নিয়ম নিচে দেওয়া হলো:

ধাপ ১ ফ্রী বা Hobby প্ল্যান সিলেক্ট করা

Vercel.com-এ গিয়ে গিটহাব দিয়ে লগইন করুন। এরপর প্ল্যান সিলেক্ট করার স্ক্রিনে অবশ্যই **I'm working on personal projects (Hobby)** সিলেক্ট করে **Continue** বাটনে ক্লিক করুন।

ধাপ ২ গিটহাব কানেক্ট বাটন

নতুন প্রজেক্ট তৈরি করার পেজে গিয়ে বাঁদিকের **Import Git Repository** এর নিচে থাকা কালো রঙের **Continue with GitHub** বাটনে ক্লিক করুন।

ধাপ ৩ গিটহাব পারমিশন দিন

গিটহাবের অথরাইজেশন পেজে সবুজ রঙের **Install** বাটনে ক্লিক করে Vercel-কে আপনার প্রজেক্ট রিড করার পারমিশন দিন।

ধাপ ৪ প্রজেক্ট ইম্পোর্ট করা

পারমিশন দেওয়া হয়ে গেলে প্রজেক্ট লিস্টে আপনার তৈরি করা **display-Adds** রিপোজিটরিটি দেখতে পাবেন। সেটির পাশে থাকা কালো রঙের **Import** বাটনে ক্লিক করুন।

ধাপ ৫ ডেপ্লয় (Deploy) বাটন ক্লিক করা

কনফিগারেশন পেজে google analytics, build command ইত্যাদি সবকিছু ডিফল্ট রেখে সরাসরি নিচে থাকা কালো রঙের **Deploy** বাটনে ক্লিক করুন।

ধাপ ৬ ডেপ্লয়মেন্ট সফল হওয়া এবং লাইভ লিঙ্ক পাওয়া

বিল্ড প্রসেস শেষ হলে স্ক্রিনে কনগ্রাচুলেশন অ্যানিমেশন দেখাবে। প্রজেক্টের ওপর ক্লিক করলেই আপনি আপনার সাইটের লাইভ ডোমেইন লিঙ্কটি পেয়ে যাবেন।

🔥 পার্ট ৩: Firebase Firestore ক্লাউড ডাটাবেস তৈরি করা

রিয়েল-টাইম তথ্য সিঙ্কের জন্য ফায়ারবেস ডাটাবেস কনফিগারেশন করার নিয়ম নিচে দেওয়া হলো:

ধাপ ১ ফায়ারবেস প্রজেক্টের নাম এন্টার করা

Firebase Console-এ গিয়ে **Add project** এ ক্লিক করার পর প্রজেক্টের নাম লিখুন **`display-adds`** এবং প্রজেক্ট ক্রিয়েট করে এগিয়ে যান।

ধাপ ২ Databases & Storage মেনু ওপেন করা

প্রজেক্টের হোমপেজ থেকে বাঁদিকের মেনুতে থাকা **Databases & Storage** অপশনটির ওপর ক্লিক করে মেনুটি বড় করুন।

ধাপ ৩ Firestore সিলেক্ট করা

মেনু থেকে **NoSQL** সেকশনের নিচে থাকা **Firestore** অপশনটির ওপর ক্লিক করুন।

ধাপ ৪ Create Database বাটন ক্লিক করা

Firestore-এর পরিচিতি পেজ লোড হলে সেখানে থাকা কমলা বর্ডারের **Create database** বাটনটিতে ক্লিক করুন।

ধাপ ৫ স্ট্যান্ডার্ড এডিশন (Standard Edition) সিলেক্ট করা

ডেটাবেসের প্রথম স্টেপে **Standard edition** সিলেক্ট করা অবস্থায় সরাসরি নিচে থাকা নীল রঙের **Next** বাটনে ক্লিক করুন।

ধাপ ৬ ডাটাবেস লোকেশন নির্ধারণ করা

পরের পেজে লোকেশনটি ডিফল্ট **nam5 (United States)** রেখে সরাসরি নিচে থাকা নীল রঙের **Next** বাটনে ক্লিক করুন।

ধাপ ৭ টেস্ট মোড (Start in test mode) সিলেক্ট করা

লোকাল অ্যাপ থেকে ফায়ারবেসে সরাসরি ডাটা রিড ও রাইট করার জন্য দ্বিতীয় অপশন অর্থাৎ **Start in test mode** সিলেক্ট করুন এবং নিচে থাকা **Create** বাটনটি ক্লিক করুন।

ধাপ ৮ ডাটাবেস প্রস্তুত হওয়া

লোডিং প্রসেস শেষ হলে আপনার ফায়ারস্টোর ডেটাবেস প্রস্তুত হয়ে স্ক্রিনে ড্যাশবোর্ড দেখাবে।

ধাপ ৯ প্রজেক্টের ওয়েব আইকন ক্লিক করা

এবার Project Overview পেজে ফিরে গিয়ে প্রজেক্টের নামের নিচে থাকা গোল আইকনগুলোর মধ্যে থেকে ওয়েব ডেপ্লয়মেন্টের জন্য **``** আইকনটিতে ক্লিক করুন।

ধাপ ১০ এপিআই কী (API Config Code) জেনারেট করা

ওয়েব অ্যাপের একটি নাম দিয়ে অ্যাপটি রেজিস্টার করলেই আপনার সামনে ফায়ারবেস কনফিগারেশনের কোড চলে আসবে।
